# Gift Certificates

`commercexl` включает подарочные сертификаты как встроенный продукт.

Что лежит в пакете:

- `GiftCertificateORM` — продукт сертификата и ссылка на целевой продукт
- `GiftCertificateOrderItemORM` — дочерний item заказа с ключом активации
- `GiftCertificateUsageORM` — факт использования ключа пользователем
- `GiftCertificateProductService` — подключение сертификата в каталог
- `GiftCertificate` — каталог и активация ключа

Что важно:

- при покупке сертификата целевой продукт не выдаётся сразу
- после оплаты сохраняется `key`
- при активации вызывается `apply_product_effect(...)` для целевого продукта
- новый заказ при активации не создаётся
